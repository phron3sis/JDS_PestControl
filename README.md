## JDS Pest Control

**JDS Pest Control** is a lightweight, loop-friendly Pest Control script designed to sit at the Knight and **clear Shifters efficiently**.

### Antiban
Antiban is a major focus in this script:
- For every possible decision an account unique trait is generated
- Attack prioritization is unique per account
- TFatigue system creates unique fatigue per account
- LOTS of per-game and per-session flags are set to diversify choices
- Weibull, LogNormal, ExpoGaussian and BetaSampling controls distribution for actions/choices

### Features
- **Automatic boat selection**, plank crossing, and movement to the Knight  
- Continuous **Shifter clearing** with stable behavior  
- Built-in **HUD** displaying wins, points, XP, and total runtime  
- Designed for long, unattended sessions
