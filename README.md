## JDS Pest Control

**JDS Pest Control** is a lightweight, loop-friendly Pest Control script designed to sit at the Knight and **clear Shifters efficiently**.

### Features
- **Automatic boat selection**, plank crossing, and movement to the Knight  
- Continuous **Shifter clearing** with stable behavior  
- Built-in **HUD** displaying wins, points, XP, and total runtime  
- Designed for long, unattended sessions
